Para usar whatsapp en tu dispositivo:
  1. Abre tu whatsapp en tu telefono.
  2. Toca el icono de menu o Configuracion y Selecciona whatsapp web.
  3. Cuando se aparesca la camara, apunta el telefono hasta esta pantalla para escanear el codigo QR.
